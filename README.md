# Leetcode Rating Extension

Displays difficulty rating and level for LeetCode problems.

## Features
- Fetches and displays problem difficulty ratings from live contest data.
- Displays the problem's level (1-10) directly on the problem page.
- Integrates seamlessly with the LeetCode dark mode UI.

## Data Sources
- [zerotrac/leetcode_problem_rating](https://github.com/zerotrac/leetcode_problem_rating)
- [stormsunshine/LeetCode-Levels](https://github.com/stormsunshine/LeetCode-Levels)
